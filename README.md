# CtrlR — Sistema de Rifas

Aplicación web para gestionar rifas: venta de números, panel de administración y sorteo del ganador.

---

## Funcionalidades

- Visualización de números disponibles y ocupados
- Panel de administración para gestionar participantes
- Sorteo automático con modal de ganador
- Generación y copia de datos al portapapeles
- Fechas en formato legible en español

## Uso

Abre `html/rifa.html` para la vista pública o `admin.html` para el panel de administración.
No requiere instalación ni servidor — HTML/CSS/JS puro.
